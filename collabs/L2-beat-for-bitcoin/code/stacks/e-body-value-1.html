Description

Stacks is a Bitcoin sidechain that is designed to provide programmability and more use cases for BTC the asset. It is a separate blockchain that uses the Proof of Transfer (PoX) consensus mechanism. It uses a smart contracting language called "Clarity" to enable developers to build a wide variety of on-chain applications. Currently, Stacks uses a federated bridge to transfer BTC to the Stacks blockchain. Transactions are mined by miners who spend BTC to eligible Stacks addresses that participate in PoX consensus.

insert-answer

Stacks full nodes and Bitcoin full nodes are able to verify the entire state of the Stacks blockchain.

Data availability

insert-answer

The data for Stacks' state updates is stored on both the Stacks and Bitcoin blockchains (Only hashes of headers are stored on Bitcoin).

Upgradeablilty

insert-answer

Community governance - Upgrading the Stacks blockchains happens through a token-weighted governance voting process.

Block producer failure

insert-answer

If your block is not mined, you are stuck in the Layer 2 and cannot force your transaction through to the Layer 1.

BTC Bridge

insert-answer

Custodial - The current Bitcoin bridge between Bitcoin and Stacks is operated by a federated group of X signers. Users' Bitcoin can be stolen if a majority of these signers collude with each other.
