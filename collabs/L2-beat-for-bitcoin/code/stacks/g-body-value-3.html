Verification

None

Bitcoin does not verify Stacks transactions.

Consensus

Proof-of-Transfer

Stacks consensus is integrated with Bitcoin consensus.

Bitcoin Hashpower

100%

Stacks transaction finality and block ordering is secured by 100% of Bitcoin hashpower.

Alternative Token

STX

STX is the native token of Stacks, used for gas fees and consensus.

Technology

State correctness

Stacks validators enforce state correctness.

Verification

Stacks validators are solely responsible for verificaiton.

Data availability

Stacks uses Bitcoin for data availability.

Operator

Open

Any user can run a node or validator in Stacks' permissionless system.

Withdrawals

Reliant on Stacks validators.

Users cannot submit their own withdrawal in the event of validator network failure.

Source Code

Open-source

Stacks is open-source.
