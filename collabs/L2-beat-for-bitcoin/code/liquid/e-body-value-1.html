Risk Analysis

State validation

Offchain

The state of the Liquid network is not validated by Bitcoin full nodes. It is validated by a fixed group of functionary members who were assigned these permissions on launch.

Data availability

Offchain

Liquid state updats are not stored on Bitcoin.

Upgradeablilty

Multi-sig

Permissioned signers can upgrade the system and revoke access to funds at any point.

Block producer failure
Cannot access funds

If centralized block production in the Layer 2 fails, or your transaction is censored, you are stuck in the Layer 2 and cannot access funds or withdrawal.

BTC Bridge

Federated

Permissioned nodes used to secure the bridge contract custodying funds for users.
